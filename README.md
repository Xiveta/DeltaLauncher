🚀 What is Delta Online?

Delta Online is a community-driven project that brings Gears of War 3 to PC with updated features, performance improvements, and active multiplayer support.
It is not an official product of Microsoft or The Coalition.

Key features include:

Full keyboard and mouse support
Unlocked framerate (120+ FPS)
4K resolution and graphical enhancements
New maps, characters, and skins
Classic and additional game modes (Horde, Beast, PvP, etc.)
Expanded progression and customization systems
