🚀 What is Delta Online?

Delta Online is a community-driven project that brings Gears of War 3 to PC with updated features, performance improvements, and active multiplayer support.

It is not an official product of Microsoft or The Coalition.

Key features:
Full keyboard and mouse support
Unlocked framerate (120+ FPS)
4K resolution and graphical enhancements
New maps, characters, and skins
Classic and additional game modes (Horde, Beast, PvP, etc.)
Expanded progression and customization systems
🧩 About This Launcher

This launcher provides a centralized tool to:

📦 Download and install the game client (via official sources)
🔄 Automatically manage updates (official update channels)
⚙️ Configure game settings (Proton / compatibility tools)
▶️ Launch Delta Online (shortcuts & integration)
🧱 Install additional content packages (through official launcher sources)
🛠️ Features
Automated installation of game files and dependencies
Built-in update system
Lightweight and user-friendly interface
Optional Steam integration (non-official)
Content and file management tools
Designed for future expansion
📥 Requirements
Linux system
Internet connection
Steam account and client
Proton or compatible compatibility layer (e.g. Wine)
Official Delta Online client (downloaded from official sources)

Note: This launcher does not provide or include the game client. Users must obtain Delta Online through its official distribution channels.
